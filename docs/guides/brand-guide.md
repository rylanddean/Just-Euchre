# Just Euchre — Brand Guide

> Last updated: March 2026

---

## 1. Brand Essence

**Just Euchre** is a daily Euchre card game for iOS. The name says it all: no fluff, no accounts, no ads — just the best card game ever made, once a day.

### Mission
Give Euchre players the perfect daily ritual: one focused game that respects their time and intelligence.

### Brand Promise
Every day, one great game. Nothing else.

---

## 2. Brand Personality

| Trait | Description |
|-------|-------------|
| **Focused** | Singular purpose. One game, done well. No feature bloat. |
| **Confident** | Doesn't need to explain itself. Euchre fans already know. |
| **Honest** | No dark patterns, no manipulation, no upsells. |
| **Minimal** | Strips away everything that isn't the game. |
| **Understated cool** | Premium feel without pretension. Like a well-worn deck of cards. |

---

## 3. Voice & Tone

### Principles

- **Short sentences.** The game speaks for itself.
- **No exclamation points.** Confidence doesn't shout.
- **Plain language.** No jargon, no fluff, no buzzwords.
- **Direct.** Tell people what it is. Don't over-explain.
- **A little dry.** Humor is welcome, but never forced.

### Do / Don't

| Do | Don't |
|----|-------|
| "One game a day." | "Play exciting daily Euchre challenges!" |
| "No accounts. No ads." | "We respect your privacy and never show intrusive advertising!" |
| "The best card game. Once a day." | "Experience the ultimate Euchre journey!" |
| "Your streak is over." | "Oh no! You lost your streak 😢" |
| "How to play" | "Learn the basics and master the game!" |

### Taglines (approved)

- **Primary:** One game a day.
- **Secondary:** No frills. Just the best card game ever made.
- **Alt:** The best card game. Once a day.

## 5. Color Palette

The palette is dark, precise, and high-contrast. It draws from the feel of playing cards at night — deep navy backgrounds, clean white text, and a single vivid accent that signals success and interactivity.

### Core Colors

| Name | Hex | Role |
|------|-----|------|
| **Background** | `#080B12` | Page/app background |
| **Surface** | `#1A212C` | Cards, panels, elevated surfaces |
| **Pill** | `#161C26` | Subtle containers, pill buttons |
| **Mint** | `#52F6AA` | Primary accent — CTAs, highlights, win states |
| **Red** | `#FB595E` | Hearts/diamonds suits, loss indicators |
| **Gold** | `#FDD758` | Crown, achievements, special callouts |
| **White** | `#FFFFFF` | Primary text, foreground |
| **Muted** | `#B8B8B8` | Secondary/supporting text |
| **Subtle** | `#8C8C8C` | Tertiary text, footer, placeholders |
| **Border** | `#474747` | Dividers, outlines, separators |

### Accent Usage Rules

- **Mint** is the only interactive accent color. Use it for primary CTAs, active states, streak indicators, and win confirmations.
- **Red** is reserved for Hearts/Diamonds suit icons and loss/error states. Do not use as a general accent.
- **Gold** is reserved for crown/achievement iconography. Do not overuse.
- Never use mint and gold together as competing accents in the same component.

### Dark Mode Only

Just Euchre is a dark-mode product. There is no light mode. The palette is optimized for low-light contrast and OLED displays. Do not create light-mode variants.

## 6. Typography

### Typeface

Just Euchre uses the **system sans-serif stack** across all platforms:

```
-apple-system, system-ui, "Helvetica Neue", sans-serif
```

This means San Francisco on Apple platforms. No custom typefaces are used or needed — the system font is part of the brand identity (native, clean, familiar).

### Type Scale

| Role | Size | Weight | Notes |
|------|------|--------|-------|
| Hero headline | 38–56px (fluid) | 700 | Letter-spacing: -1.5px |
| Section title | 26–36px (fluid) | 700 | |
| Card title | 18–22px | 600 | |
| Body | 16px | 400 | Line-height: 1.5 |
| Caption / label | 12px | 700 | Uppercase, letter-spacing: 1.5px |
| Subtitle | 13–14px | 500 | |

### Rules

- Headlines use tight letter-spacing (negative tracking) to feel precise and confident.
- Labels are always uppercase with tracked-out letterspacing — they function as category markers, not text.
- Body text is never smaller than 14px.
- Line lengths should stay between 50–75 characters for readability.

## 7. Iconography & Symbols

### Suit Symbols

The four suit symbols (♠ ♥ ♦ ♣) are core brand marks. They appear in:
- The footer of every web page
- Suit indicators throughout the game UI
- Marketing copy as typographic flourishes

**Color mapping:**
- ♠ Spades — White
- ♣ Clubs — White
- ♥ Hearts — Red (`#FB595E`)
- ♦ Diamonds — Red (`#FB595E`)

### App Icon

The app icon is the single canonical brand mark. It should always:
- Appear at its designed aspect ratio (1:1 square)
- Use the standard iOS squircle mask when shown in-app or in the nav
- Never be recolored, stretched, or placed on a light background

### Crown Symbol (👑)

The crown is used exclusively for the "loner" mechanic (going alone / loner mode). It is a functional game symbol, not a general decorative element.

---

## 8. Photography & Screenshots

Marketing screenshots show the actual app UI. They should:
- Use real in-game states (not staged or idealized data)
- Be captured on a device with a dark wallpaper/background
- Never include personal identifying information
- Showcase the game in progress, the home screen, streak history, or settings

Do not use stock photography, lifestyle imagery, or illustrated characters.

---

## 9. What Just Euchre Is Not

To stay on-brand, avoid the following:

- **Not casual/hyper-casual.** No bright rainbow palettes, no bouncy animations, no cartoon characters.
- **Not gamified beyond the game.** No points systems, leaderboards, or social pressure mechanics.
- **Not corporate.** No mission statement language, no "we're on a journey", no buzzwords.
- **Not social.** No sharing prompts, no friend features, no "beat your friends" messaging.
- **Not aggressive.** No push notifications designed to re-engage, no streak-loss panic mechanics.
